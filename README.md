# GoblinMaster
GoblinMaster es una aplicación web desarrollada como proyecto final de la Tecnicatura Universitaria en Programación (UTN - Facultad Regional Mendoza). Su objetivo es brindar una herramienta integral para Dungeon Masters (directores de juego) y jugadores en partidas de rol de mesa, especialmente aquellas basadas en el sistema Pathfinder.  El backend, desarrollado en Java con Spring Boot, se encarga de gestionar la lógica principal del sistema, la autenticación de usuarios, y la comunicación con las bases de datos. La arquitectura se diseñó para combinar MySQL (para datos estructurados como usuarios, personajes y combates) y MongoDB (para información más flexible, como bestiarios, dotes y reglas).
